0 Страница спецификации FHIR®

FHIR - это стандарт обмена медицинскими данными, опубликованный HL7®.

Впервые здесь?
Для ознакомления с FHIR посмотрите: Краткое описание, Введение для программистов, Введение для медицинских работников, Введение для архитекторов ПО, Обзор стандарта FHIR и план развития, История версий. См. также открытую лицензию, оглавление, благодарности сообществу и поиск по спецификации.
