4.4.2.1097 HL7 v3 Value Set HL7StandardVersionCode

Vocabulary Work Group Maturity Level: N/AExternal Use Context: Any

This value set (http://terminology.hl7.org/ValueSet/v3-HL7StandardVersionCode) is defined as part of HL7 v3.

Summary

Defining URL:http://terminology.hl7.org/ValueSet/v3-HL7StandardVersionCode
Version:2018-08-12
Name:v3.HL7StandardVersionCode
Title:v3 Code System HL7StandardVersionCode
Definition:

This code system holds version codes for the Version 3 standards. Values are to be determined by HL7 and added with each new version of the HL7 Standard.

OID:2.16.840.1.113883.1.11.19373 (for OID based terminology systems)
Source ResourceXML / JSON

This value set is used in the following places:

  • CodeSystem: This value set is the designated 'entire code system' value set for v3.HL7StandardVersionCode

  • Include all codes defined in http://terminology.hl7.org/CodeSystem/v3-HL7StandardVersionCode
