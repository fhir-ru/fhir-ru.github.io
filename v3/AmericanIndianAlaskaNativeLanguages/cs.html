Vocabulary Work Group Maturity Level: N/AExternal Use Context: Any

This code system (http://terminology.hl7.org/CodeSystem/v3-AmericanIndianAlaskaNativeLanguages) is defined as part of HL7 v3.

Summary

Defining URL:http://terminology.hl7.org/CodeSystem/v3-AmericanIndianAlaskaNativeLanguages
Version:2018-08-12
Name:v3.AmericanIndianAlaskaNativeLanguages
Title:v3 Code System AmericanIndianAlaskaNativeLanguages
Definition:

American Indian and Alaska Native languages currently being used in the United States.
