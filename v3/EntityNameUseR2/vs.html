4.4.2.1120 HL7 v3 Value Set EntityNameUseR2

Vocabulary Work Group Maturity Level: N/AExternal Use Context: Any

This value set (http://terminology.hl7.org/ValueSet/v3-EntityNameUseR2) is defined as part of HL7 v3.

Summary

Defining URL:http://terminology.hl7.org/ValueSet/v3-EntityNameUseR2
Version:2018-08-12
Name:v3.EntityNameUseR2
Title:v3 Code System EntityNameUseR2
Definition:

Description: A set of codes advising a system or user which name in a set of names to select for a given purpose.

OID:2.16.840.1.113883.1.11.20321 (for OID based terminology systems)
Source ResourceXML / JSON

This value set is used in the following places:
